## Visão Geral

Este repositório documenta um estudo conceitual de geometria auxética para motocicletas. Os parâmetros foram explorados usando IA como ferramenta de design generativo, com ~10.000 iterações de simulação numérica. 

**Todos os valores são resultados teóricos de simulação e NÃO foram validados experimentalmente.**

## ⚠️ Disclaimer Técnico e de Segurança

**Projeto de pesquisa conceitual em estágio inicial. Não validado para uso real.**

1. **Não testado**: Nenhum teste físico, de impacto ou certificação foi realizado. Dados são teóricos de simulação.
2. **Não usar em produção**: NÃO fabricar equipamentos de proteção real com base neste repo sem validação experimental por profissionais qualificados.
3. **Método**: Geometria explorada com assistência de IA Muse Spark e simulação numérica. Código e método serão publicados na v1.1 para reprodutibilidade.
4. **Sem garantia**: Sem garantia de desempenho ou segurança. Uso por conta e risco do usuário.

O objetivo é fomentar pesquisa aberta, não fornecer produto acabado.
