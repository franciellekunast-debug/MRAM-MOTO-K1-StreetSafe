## Visão Geral

Este repositório documenta um estudo conceitual de geometria auxética para motocicletas. Os parâmetros foram explorados usando IA como ferramenta de design generativo, com ~10.000 iterações de simulação numérica. 

**Todos os valores são resultados teóricos de simulação e NÃO foram validados experimentalmente.**

2. **Não usar em produção**: NÃO fabricar equipamentos de proteção real com base neste repo sem validação experimental por profissionais qualificados.
